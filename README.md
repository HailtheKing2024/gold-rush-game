# Gold Rush Prototype

This repo contains prototype files for **Gold Rush**, a game currently in active development. Things may be unfinished, broken, or change frequently. There is no guarantee that everything will work — this is a work in progress.

This project is a fan-made remake of _The Oregon Trail_. It is being built completely from scratch.

## Status

- 🚧 In development
- 🧪 Prototype build
- 🔄 Frequent updates and changes

The final version can be found on https://gold-rush-game.onrender.com/
